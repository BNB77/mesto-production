## Mesto

Интерактивный сервис для работы с фотографиями. Пользователи могут добавлять карточки с фотографиями, ставить лайки и редактировать профиль.

### Технологии
- JavaScript (ES6 modules)
- Vite
- REST API

### Запуск проекта
npm run dev

### Сборка
npm run build

### Публикация
npm run deploy
